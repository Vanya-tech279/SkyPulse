# SkyPulse Weather App
SkyPulse is a responsive website that enables users to check the weather for any city or their current location.
## Features
- Enables users to search for weather by city name with input validation.
- Enables users to get weather information using their current location (geolocation).
- Provides a dropdown of the last 5 searched cities for quick access.
- Displays dynamic backgrounds that change based on weather conditions and day/night.
- Includes eight highlight cards showing weather details: Pressure, Humidity, Clouds, Wind Speed, Visibility, Feels Like, Sunrise, Sunset, each with icons.
- Enables users to toggle temperature display between Celsius and Fahrenheit.
- Shows friendly error messages for invalid input, denied location access, or API errors.
- Fully responsive design optimized for desktop, tablet, and mobile devices.
- Hides sidebar on mobile and shows it on desktop for better usability.
- Layout grid adjusts based on screen size:
  - Mobile: 1 column
  - Tablet: 2 columns
  - Desktop: 3 columns
- Includes smooth animations and transitions for UI elements and background changes.

## Technologies Used
- HTML5
- Tailwind CSS
- JavaScript (ES6)
- OpenWeatherMap API

## How to Run
1. Clone the repository to your local machine.
2. Open `index.html` in any modern web browser.
3. Ensure your computer is connected to the internet to fetch real-time weather data.

## API Information
- Weather data is retrieved from the **OpenWeatherMap API**.
- Provides real-time weather data, including temperature, wind, humidity, and 5-day forecasts.

## Responsiveness
The app is fully responsive and enables smooth usage on:
- Desktop
- iPad Mini
- iPhone SE
